# Documentation Table of Contents

This directory contains all project documentation, structured as a deep tree. Each subdirectory has a README.md as its TOC.

## Subdirectories
- `architecture/`: System design (server, client, network, persistence).
- `protocol/`: Networking protocol (messages, versioning).
- `gameplay/`: Game mechanics (systems, content, balance).
- `operations/`: Deployment and maintenance (Docker, monitoring).
- `decisions/`: Architectural decisions (ADRs).

Delete unused docs entirely.
