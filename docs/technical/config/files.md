# Config File Map

All files are JSON under `config/`.

- `server.json`: HTTP/WebSocket ports, tick rate, rate limits.
- `world.json`: chunk size, streaming radii, world seed.
- `balance.json`: player/mob stats, item scaling, base damage.
- `survival.json`: hunger, temperature, thirst, regen rates.
- `crafting.json`: recipe tables and station tiers.
- `spawning.json`: spawn budgets, respawn timers, density caps.
- `biomes.json`: biome definitions and modifiers.
- `settlements.json`: barrier cores, settlement tiers, guard rules.
- `economy.json`: vendor pricing, taxes, currency sinks.
- `quests.json`: quest templates and reward tables.
- `achievements.json`: achievement definitions and bonuses.
