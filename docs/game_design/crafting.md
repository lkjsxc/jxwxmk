# Crafting System

Crafting allows players to convert raw resources into useful items.

## Workstations

- **Hand**: Basic items (Campfire, Wood Pickaxe).
- **Workbench**: Standard tools and structures.
- **Furnace**: Smelting ores.

## Resources

- **Wood**: Trees. Basic building block.
- **Stone**: Rocks. Tools and walls.
- **Gold**: Rare rocks. Scoring/Currency.
- **Diamond**: Very rare. High-tier gear.
- **Meat**: Animals. Food.
- **Fur**: Animals. Winter gear.

## Inventory

- Fixed slots (e.g., 10 slots).
- Stackable items (max 255).
- Quick bar for usage.
