# Trees

Each tree type maps to a level tier and biome affinity.

- **Birch (L1)**: meadow/forest, light logs.
- **Spruce (L2)**: boreal, resin-rich logs.
- **Pine (L2)**: boreal, softwood logs.
- **Willow (L2)**: swamp, flexible fibers.
- **Oak (L3)**: forest, hardwood logs.
- **Maple (L3)**: forest, syrup sap.
- **Cedar (L4)**: mountain, durable beams.
- **Ash (L5)**: savanna, dense wood.
- **Redwood (L6)**: jungle, ancient timber.
- **Baobab (L7)**: savanna, massive trunks.
- **Ironwood (L8)**: swamp, ironwood logs.
- **Ebon (L10)**: volcanic, ebonwood cores.

## Level Rules

- Required tool tier >= tree level.
- Yield increases with level; respawn cooldown scales upward.
