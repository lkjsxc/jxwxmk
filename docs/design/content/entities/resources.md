# Resources

Static entities harvestable by players.

## Types

### Tree
- **HP**: 5 hits (Hand), 2 hits (Axe).
- **Drops**: 3-5 Wood.
- **Respawn**: 60 seconds.

### Rock
- **HP**: 10 hits (Hand), 3 hits (Pickaxe).
- **Drops**: 2-4 Stone.
- **Respawn**: 120 seconds.
- **Requirement**: Better to use Pickaxe.

### Bush
- **HP**: 1 hit.
- **Drops**: 1-3 Berries.
- **Respawn**: 30 seconds.

### Gold Node
- **HP**: 8 hits (Pickaxe only).
- **Drops**: 1-2 Gold.
- **Rarity**: High.
