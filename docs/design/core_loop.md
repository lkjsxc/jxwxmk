# Core Loop

The game follows a standard survival sandbox loop.

## Cycle

1.  **Gather**: Collect basic resources (wood, stone, food) from the environment.
2.  **Survive**: Manage hunger and temperature (cold/heat).
3.  **Craft**: Create tools, weapons, and structures to improve efficiency and defense.
4.  **Build**: Establish a base to protect against mobs and other players.
5.  **Dominate**: Upgrade gear, control territory, and defeat high-level threats.

## Progression

- **Early Game**: Struggle for food and warmth. Avoid combat.
- **Mid Game**: Establish a secure base. Farm resources. basic PVP.
- **Late Game**: High-tier gear (diamond/amethyst). Clan wars. Boss fights.
