# Core Loop

The game follows an MMORPG survival loop with long-term progression and social play.

## Cycle

1. **Spawn + Orient**: Bind to a settlement and learn local rules.
2. **Gather**: Harvest leveled resources to craft and trade.
3. **Survive**: Manage hunger, temperature, and biome hazards.
4. **Craft**: Progress through tool, weapon, and armor tiers.
5. **Build**: Improve personal shelter and contribute to settlement growth.
6. **Quest + Explore**: Clear local quest lines, unlock new biomes and towns.
7. **Compete + Cooperate**: Group up for events, bosses, and trade routes.

## Progression Bands

- **Early Game**: Starter settlement, basic tools, local quests.
- **Mid Game**: Multi-biome travel, advanced crafting, faction reputation.
- **Late Game**: World events, elite bosses, settlement leadership, rare materials.
