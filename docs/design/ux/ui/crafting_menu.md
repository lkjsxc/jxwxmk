# Crafting Menu

Interface for creating new items.

## Layout
-   **List View**: Scrollable list of known recipes on the left.
-   **Detail View**: Selected recipe details on the right.
    -   Required materials (Red if missing, Green if present).
    -   "Craft" Button.
    -   "Craft All" Button.

## Categorization
Tabs for easy navigation:
-   **Tools** (Pickaxes, Swords)
-   **Structures** (Walls, Doors, Workbenches)
-   **Consumables** (Food, Bandages)
-   **Misc** (Torches, Clothing)
