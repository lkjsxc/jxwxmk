# Main Menu

The entry point of the game.

## Screens

1.  **Login / Register**:
    -   Username / Password fields.
    -   "Play as Guest" option.
2.  **Server List** (Future proofing):
    -   List of available game servers with player counts.
3.  **Settings**:
    -   Volume (SFX, Music).
    -   Graphics (Low/High - disable particles).
    -   Controls (Sensitivity).
4.  **Death Screen**:
    -   "You Died".
    -   Stats: Time Survived, Score.
    -   "Respawn" Button.
