# Main Menu (The Hub)

The central interface for the game. It serves as both the **Start Screen** and the **In-Game Pause/Management Menu**.

## States

### 1. Start Screen
Displayed on load.
- **Play**: Connects to server and spawns player.
- **Settings**: Audio/Graphics/Controls.
- **Guidebook**: Read-only tutorial.
- **Credits**: Developer info.

### 2. In-Game Hub
Opened via 'E' (Desktop) or Menu Button (Mobile).
- **Tabs**:
    1.  **Inventory**: Large bag grid (Main storage).
    2.  **Crafting**: Recipe list and creation button.
    3.  **Guidebook**: Reference manual.
    4.  **Settings**: Adjust on the fly.
-   **Resume**: Close menu.
-   **Disconnect**: Return to Start Screen.

## Design
-   **Visual**: Full-screen overlay with a semi-transparent background (if in-game).
-   **Navigation**: Tab bar at the top for switching contexts.
