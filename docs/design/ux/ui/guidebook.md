# Guidebook

A comprehensive in-game manual accessible via the Main Menu.

## Structure
The guidebook is divided into chapters:

### 1. Basics
- **Controls**: Movement, Interaction, Combat.
- **HUD**: Explaining Vitals (Health, Hunger, Cold).

### 2. Survival
- **Gathering**: How to get Wood, Stone, Food.
- **Crafting**: How to use the Crafting tab.
- **Building**: Placing walls and doors.

### 3. Bestiary
- **Passive**: Rabbits (Food source).
- **Hostile**: Wolves, Bears (Combat tactics).

### 4. Items
- List of all tools and their tier (Wood < Stone < Gold < Diamond).
- Consumables effects.
