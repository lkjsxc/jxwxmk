# Desktop Controls

Desktop uses the same interaction rules as mobile, with keyboard shortcuts retained as optional accelerators.

## Key Map

- **W/A/S/D**: Movement.
- **Mouse Move**: Aim/cursor.
- **Left Click (Tap)**: Primary action (attack/gather).
- **Click-and-Hold (Long-Press)**: Interact (open/use/talk).
- **E**: Interact (keyboard shortcut).
- **Number Keys (1-7)**: Select hotbar slot.
- **Enter**: Chat.
- **M**: Map.

## HUD

- **Hotbar**: Always visible at the bottom of the screen (centered).
- **Vitals**: Top-left corner bars.
- **Minimap**: Bottom-right (optional).
