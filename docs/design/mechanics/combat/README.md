# Combat Mechanics

PVP and PVE systems.

## Damage Calculation
`Damage = BaseDamage * (1 - (Defense / 100))`

## Weapons
- **Hand**: 2 DMG.
- **Wood Pickaxe**: 4 DMG.
- **Stone Sword**: 10 DMG.
- **Diamond Spear**: 25 DMG.

## States
- **Knockback**: 5px push on hit.
- **Invincibility Frames**: 0.2s after hit.
