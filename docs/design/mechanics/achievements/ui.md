# Achievement UI Design

## Display
- **List View**: Displays **ALL** achievements, both locked and unlocked.
    - **Unlocked**: Full color icon/text.
    - **Locked**: Grayed out or silhouette, maybe with "???" if secret (but for now show all).
- **Selection**: Clicking an achievement selects it.

## Details Pane
- When an achievement is selected, show a details pane (side or popup).
- **Content**:
    - **Title**: Name of the achievement.
    - **Description**: How to unlock it.
    - **Status**: "Unlocked" (Date?) or "Locked".
    - **Progress**: Current progress bar (e.g., "Mobs Killed: 45/100").
    - **Reward**: Stat bonus description.
