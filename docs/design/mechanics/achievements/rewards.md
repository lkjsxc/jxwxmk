# Rewards & Notifications

## Stat Bonuses

Achievements grant additive bonuses that are applied server-side:

- `speed`
- `damage`
- `gather`
- `max_hp`
- `craft`
- `resistance`
- `reputation_gain`

## XP Rewards

- Achievements grant a one-time XP award on completion.
- XP feeds the player level system and is persisted with the character.

## Reward Types

- Flat stat bonuses.
- XP grants.
- Cosmetic titles.
- One-time item bundles.

## Notifications

- Unlocks trigger a toast notification.
- Duration: 3-5 seconds.
- Content: title + achievement name + optional reward summary.
