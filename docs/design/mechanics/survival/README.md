# Survival Mechanics

Survival is the primary loop. Players must maintain their vitals.

## Vitals

- **Health (HP)**: 0-100.
- **Hunger**: 0-100.
- **Temperature**: 0-100 (Where 0 is Freezing, 100 is Overheat).
- **Thirst**: 0-100.
