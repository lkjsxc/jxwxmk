# kkmypk

A real-time survival game inspired by starve.io, built with Rust and TypeScript.

## Project Structure
- `src/`: Source code for backend (Rust) and frontend (TypeScript).
- `docs/`: Technical documentation and architecture details.
- `AGENTS.md`: Development guidelines for agentic coding.

## Quick Start
1. Ensure Docker Compose is installed.
2. Run `docker-compose up -d` to start the infrastructure.
3. Refer to `AGENTS.md` for build and test commands.
