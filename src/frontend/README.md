# TypeScript Frontend

The frontend handles game rendering, user input, and real-time communication with the server.

## Architecture

- **index.html**: Main HTML file
- **main.ts**: Application entry point
- **components/**: UI components and game objects
- **services/**: Network and utility services
- **utils/**: Helper functions and math utilities
- **assets/**: Game assets (images, sounds)

## Key Features

- HTML5 Canvas rendering
- Real-time WebSocket client
- Component-based UI system
- Asset management and loading
- Input handling and game controls
