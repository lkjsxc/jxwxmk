# Frontend Implementation

TypeScript-based game client for kkmypk.

## Technologies
- **Language:** TypeScript
- **Rendering:** HTML5 Canvas / WebGL
- **State Management:** Reactive stores
- **Networking:** WebSockets

## Structure
- `src/core/`: Game engine and rendering logic.
- `src/ui/`: UI components and overlays.
- `src/network/`: WebSocket client and protocol handling.
