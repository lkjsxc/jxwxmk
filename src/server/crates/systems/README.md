# Systems Crate

Deterministic gameplay systems invoked by the tick loop.

## Modules

- `survival.rs`: Hunger, temperature, healing
- `interaction.rs`: Movement, targeting, gather/attack
- `crafting.rs`: Recipe validation and consumption
- `spawning.rs`: Chunk budgets and respawn timers
- `barriers.rs`: Safe-zone checks
- `death.rs`: Death detection and respawn
- `achievements.rs`: Stat-based achievement grants
- `quests.rs`: Quest accept and progress
