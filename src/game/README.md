# Game Simulation

Authoritative world state, fixed tick loop, and gameplay systems.

## Contents

- `mod.rs`: exports and shared types.
- `engine/`: `GameEngine` actor + tick loop.
- `events/`: input and command events.
- `world/`: world, chunks, and entity data.
- `systems/`: survival, interaction, crafting, combat, quests, achievements, barriers, death.
