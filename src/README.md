# Source Code

This directory contains the authoritative source code for the game, including the Rust backend and the TypeScript frontend.

## Directory Structure

- `client/`: TypeScript frontend application.
- `server/`: Rust server and networking logic.
- `game/`: Authoritative game simulation logic.
- `db/`: Database models and persistence layer.
- `runtime/`: Operational configuration, including the Dockerfile.
- `shared/`: (Optional) Shared logic or protocol definitions.
