# UI Components

Canvas-rendered and DOM-based UI components for the game client.

## Components

- `manager.ts` - UIManager coordinating all UI
- `hud.ts` - HP/hunger/temperature bars
- `hotbar.ts` - 7-slot hotbar with active slot highlighting
- `inventory.ts` - 30-slot inventory grid
- `crafting.ts` - Crafting menu with recipes
- `quests.ts` - Quest log
- `achievements.ts` - Achievements tab
- `notifications.ts` - Toast notifications
- `screens.ts` - Login/game over overlays
- `profile.ts` - Profile page with stats
